Steps To Run  Nodejs StressTool.

Step1:Open Command Prompt in Folder Containing Project.

Step2:Check that you have node and npm installed
To check if you have Node.js installed, run this command in your terminal:
node -v
To confirm that you have npm installed you can run this command in your terminal:
npm -v

Step 3:If your System have node and npm installed skip step 3, otherwise go to https://nodejs.org/en/download/ link to Download node and npm corresponding to your system(e.g.-Windows,Linux etc).

Step 4:Run this command in your terminal to install all node modules which are required to run Stress Tool project.This command use package.json file to install all required node modules.
npm install 

Step 5:Open the folder containing project and check if folder with name node_modules(containing all required modules) is created or not.
